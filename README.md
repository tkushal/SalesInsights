# SalesInsights
Analysis on sales of atliq.

Atliq provides the products to it's customers all over India.
Our dataset in .sql format, which is been used by cleaning and modifying some feature set according to our requirements.

Data is provided within the folder MySQL.

Observations:
1. Drawn out Revenue analysis based on the total sales quantity with repect to every regions of India.
2. Most revenue generating Market is been plotted using graph.
3. Drawn out sales details that signifies which Market has most number of sales.
4. Top 5 customers is been drawn out in which we found 'Electralsala Stores' to be the top among all customer.
5. Top 5 products is been drawn out in which we found 'Product 40' is set to top all product in terms of sales.
